1. Gerekli Donanımlar

  • Kontrol Kartı: ESP32 veya HTTP isteği destekleyen röle modülü.
  • Tarayıcı: 2D (Karekod) Barkod Okuyucu.
  • Bağlantı: Internet erişimi (Wi-Fi veya LAN).

2. API Protokolü

Donanım cihazı, doğrulama için aşağıdaki uç noktaya POST isteği göndermelidir:

POST /api/validate/

Header Bilgileri:

  • X-API-KEY : .env dosyasındaki TURNSTILE_API_KEY değerini kullanın.
  • Content-Type : application/json

Gönderilecek Veri (JSON):

{ "qr_content": "ÖĞRENCİ_NO:TOKEN" }

3. Donanım Bağlantısı (Perkotek S150)

Turnike ana kartı üzerindeki Dry Contact girişlerini kullanın:

  • Kontrol kartı rölesinin COM ucunu turnike GND ucuna bağlayın.
  • Rölenin NO (Normally Open) ucunu turnike OPEN ucuna bağlayın.

4. Çalışma Mantığı

  1. Okuyucu karekodu tarar.
  2. Kontrolör, okunan metni sunucuya iletir.
  3. Sunucu doğrularsa (Success 200), status: allowed yanıtını döndürür.
  4. Kontrolör röleyi tetikler ve turnike geçişe izin verir.

۱. تجهیزات مورد نیاز

  • برد کنترلر: ESP32 یا ماژول‌های رله تحت شبکه.
  • اسکنر: ماژول بارکد اسکنر دو بعدی (QR Reader).
  • اتصال: دسترسی به اینترنت (Wi-Fi یا کابل LAN).

۲. پروتکل ارتباطی (API)

سخت‌افزار باید درخواست خود را به نقطه اتصال زیر ارسال کند:

POST /api/validate/

هدرهای مورد نیاز:

  • X-API-KEY : از مقدار TURNSTILE_API_KEY در فایل .env استفاده کنید.
  • Content-Type : مقدار application/json

ساختار داده ارسالی (JSON):

{ "qr_content": "STUDENT_NO:TOKEN" }

۳. سیم‌کشی سخت‌افزار (Perkotek S150)

از ورودی‌های Dry Contact روی برد اصلی گیت استفاده کنید:

  • پایه COM رله را به پایه GND گیت متصل کنید.
  • پایه NO رله را به پایه OPEN گیت متصل کنید.

۴. منطق عملکرد

  1. اسکنر کد QR را می‌خواند.
  2. کنترلر رشته دریافتی را به سمت سرور می‌فرستد.
  3. سرور در صورت تایید (Success 200)، پیام status: allowed را برمی‌گرداند.
  4. کنترلر رله را فعال کرده و گیت آزاد می‌شود.
